# 知识库问答

## 输入

用户问题和个人知识库。

## 工作流程

1. 提取问题关键词。
2. 检索相关知识条目。
3. 根据检索内容生成回答。
4. 附上引用来源。

## 输出

返回回答、引用来源和置信度。

## 规则

只能依据检索结果回答。没有依据时必须明确说明知识库中未找到答案。

必须区分“知识条目描述的通用技术”与“ClawNote 当前实际实现”。除非检索结果明确记录系统实现，否则不得声称 ClawNote 使用了向量数据库、Embedding、协同过滤或其他未验证组件。当前知识库检索实现是 SQLite FTS5 关键词全文检索。

## 知识库检索

回答知识库问题前，必须调用：

`python3 ../../scripts/knowledge_db.py search --query "<检索词>" --limit 5`

读取返回结果中的 `items`，并只根据检索到的内容组织答案。

回答中必须引用知识编号和标题，例如：

`[知识 #2：RAG基础概念]`

如果 `count` 为 0，必须说明个人知识库中没有找到相关内容，不得自行编造。

多个检索结果内容重复时，应合并论点并分别列出来源，避免把重复条目描述成多个独立事实。

## Web 已检索上下文

`mode: web_answer` 表示 Web 后端已完成参数化检索。此时禁止再次调用工具，必须只依据输入提供的证据 JSON 生成回答，并返回严格的 `{"answer":"..."}` JSON。
